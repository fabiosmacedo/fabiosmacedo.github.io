Pagina Sobre

Esse site é um modelo feito com:

  • bootstrap 4 - Estilizar os componentes
  • Jquery - Usando para fazer o carrementos das paginas dinamicamente
  • FontAwesome 4.7 - Icones do Site
  • GitHub - Para Versionar o codigo deixa-lo de forma colaborativa

Proposito:

  • Usar tecnicas, tags e tecnologias aprendidas em sala

Conceito usado:

  • Inovar na página de ecommerce com uma estrutura dinâmica
  • Carregar páginas sem precisar carreagar todo site, com um resultado muito mais rápido
  • Deixar os produtos em um Scroll veritical e automatico diferente de quaquer site de vendas
